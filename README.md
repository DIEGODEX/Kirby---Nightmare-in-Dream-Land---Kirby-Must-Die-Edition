# Kirby - Nightmare in Dream Land - Kirby Must Die Edition v2

A hard mode modification for **Kirby: Nightmare in Dream Land (GBA)**.  
This patch increases the difficulty by modifying enemy damage, hazards, bosses, sub-bosses, and selected regular enemies.

## Author

Created by: SuperShyguy MKW

## Installation

This release is distributed as a BPS patch only.

You need:

- A clean Kirby: Nightmare in Dream Land (USA) Copy
- The .bps patch file
- A patching tool such as:
  - Floating IPS / Flips
  - Rom Patcher JS

### Quick Install Steps

1. Open your BPS patching tool.
2. Select the clean **Kirby: Nightmare in Dream Land (USA)** ROM.
3. Select the included `.bps` patch.
4. Apply the patch.
5. Load the patched ROM in your preferred GBA emulator.

## Main Changes

- Spikes are now instant-kill hazards.
- Grab attacks from selected enemies are now instant-kill:
  - Phan Phan
  - Bugzzy
  - Fire Lion
- Kirby takes more damage from regular enemy contact (3 points).
- Boss HP values were increased.
- Sub-boss HP values were increased.
- Selected regular enemies received HP increases:
  - Sir Kibble
  - Blade Knight
  - Rocky
  - Sword Knight
  - Coner

## Notes

- This patch is designed for the **USA version** of the game.
- The goal is to make the game harder while keeping it playable.
